Specificity

A stylesheet may contain many CSS rules. A declaration in one rule may conflict with a declaration in another rule.

Example:

All text in this div will appear in green color. However, this sentence will appear in red color, and this phrase within the sentence will appear in blue color.

In the example above a conflict occurs for the "font-weight" declaration. Conflicts are resolved by considering the sepcificity of the rule. Rules with higher specificity override rules with lower specificity. Here is a list of selectors with increasing order of specificity:

  1. Type selectors ( div )
  2. Class selectors ( sentence )
  3. ID selectors ( phrase )

Cascading and Reset

Notice that sentence inherits the "font-weight: bold;" style from div. The "font-weight" declaration cascades from div into sentence. The phrase undoes the effect of cascading by applying a reset of "font-weight: initial;".
