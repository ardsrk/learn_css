Relative Positioning

The default position of an element is static. The element usually fits into the space provided by its parent.

position: static; margin-left: 50px;

With relative positioning, the element moves relative to its natural position. Its new position can be defined by setting values for top, left, bottom, and right properties. Width of a relatively positioned element is not changed to fit into the parent's box.

position: relative; left: 50px;

Inline elements too can be relatively positioned.
