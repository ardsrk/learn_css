Box Model

Assume we received a package from Amazon. According to the Box Model we would first remove the margin, then remove the border, then the padding, and finally take the content.

Content Box
Border Box

Both the boxes have width and height of 100 pixels. But the value of box-sizing property decides the actual space available for the content.

The box having content-box as the value for "box-sizing" gets to use 100 pixels of width and height for its content.

While the box having border-box as the value for "box-sizing" gets to use only 30 pixels of width and height for its content.

Note

Open the dev tools of your browser and observe that the text in the second box overflows the content and spills over to the padding area.

Also notice that text starts filling the content area starting from the top-left position.
