CSS Rule

A CSS rule is composed of selectors and declarations. Each declaration is composed of a property and a value.

Example:

blockquote {
 color: blue;
}

In the example above,

  • blockquote is a selector
  • "color: blue;" is a declaration
  • color is a property
  • blue is a value

Selectors

As the name suggests, a selector selects the elements in a page to which the declarations apply. See this MDN article on selectors for more information.
