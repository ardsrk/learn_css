Understanding Width

The default value of the width property for block elements is auto. By default, a block-element will occupy all the space provided by its parent. The div occupies 800 pixels ( including margin ), because that is the width set on the document body.

auto width

An element can choose to override the default by setting the width property. Width of div below is 400 pixels because it's width is set to 50%.

50% width

A width of 100% will make the div use 800 pixels for its content causing it to break out from the parent's bounds.

100% width
