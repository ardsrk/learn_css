Understanding Width

The default value of the width property for block elements is auto. By default, a block-element will occupy all the space provided by its parent. The div occupies 800 pixels ( including margin ), because that is the width set on the document body.

auto width

An element can choose to override the default by setting the width property. Width of div below is 400 pixels because it's width is set to 50%.

50% width

A width of 100% will make the div use 800 pixels for its content causing it to break out from the parent's bounds.

100% width

While auto consumes width provided by the parent, the min-content value consumes width based on its children.

min content takes the width needed by the lonngggest word

The value max-content is the opposite of min-content. It consumes width needed to fit its children in a single line.

Max Content never wraps.

The value fit-content never exceeds space provided by its parent and acts like auto. If the content of its children fit the width then it does not break up the contents and acts like max-content.

Fit Content fits to the availabe width
